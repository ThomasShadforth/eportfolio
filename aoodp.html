Artefacts

This first exercise focused on Inheritance, a key concept in Object-Oriented Programming

          
            class Vehicle:
                def __init__(self, brand: str, fuel_type: str) -> None:
                    self.brand = brand
                    self.fuel_type = fuel_type

            class Car(Vehicle):
                def __init__(self, brand: str, fuel_type: str, num_doors: int) -> None:
                    self.num_doors = num_doors
                    super().__init__(brand, fuel_type)

            porsche = Car("Porsche", "Diesel", 2)

            print(porsche.brand)
            print(porsche.fuel_type)
            print(porsche.num_doors)

Inheritance is a critical concept as inheriting from parent classes allow for reuse of code established in those base classes, as well as specifying implementations of class methods through method overloading
